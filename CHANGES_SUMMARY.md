## Пример работы

При получении поста с 3 фото в Telegram канал:

1. **До публикации:**
   ```
   /root/crossposting/media/
   └── 2025-04-19_22-30-15/
       ├── photo1.jpg
       ├── photo2.jpg
       └── photo3.jpg
   ```

2. **URL для Instagram:**
   ```
   https://leaduxai.id/media/2025-04-19_22-30-15/photo1.jpg
   https://leaduxai.id/media/2025-04-19_22-30-15/photo2.jpg
   https://leaduxai.id/media/2025-04-19_22-30-15/photo3.jpg
   ```

3. **После публикации:**
   ```
   /root/crossposting/media/
   └── (пусто - папка удалена)
   ```

## Преимущества

✅ **Чистота** - файлы не накапливаются на сервере  
✅ **Организация** - каждый пост в своей папке с датой  
✅ **Публичные URL** - Instagram Graph API может скачать файлы  
✅ **Безопасность** - уникальные имена папок (включая секунды)  
✅ **Автоматизация** - всё происходит автоматически

## Примечания

- Для Instagram Graph API требуется, чтобы файлы были доступны по публичному URL
- Instagram скачивает файлы самостоятельно во время публикации
- После успешной публикации файлы сразу удаляются
- Если публикация не удалась, файлы всё равно удаляются (в `finally` блоке)
