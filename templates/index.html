Как пользоваться

1
Напиши тему в канале
Просто опубликуй текст с темой или кратким описанием идеи для карусели.
2
Одобри в личке у бота
Бот пришлёт сообщение с кнопками «✅ Создать карусель» и «❌ Отклонить».
3
Gemini генерирует карусель
Gemini 2.5 Flash разбивает тему на {{ slide_count }} слайдов → Gemini Image Gen рисует картинки.
4
Готово — карусель в канале
Альбом из {{ slide_count }} слайдов автоматически публикуется в канале.

Настройка .env

TELEGRAM_BOT_TOKEN=...        # токен бота
TELEGRAM_CHANNEL_CHAT_ID=... # ID канала (-100...)
NOTIFY_CHAT_ID=...           # твой личный chat_id
GEMINI_API_KEY=...           # ключ от Google AI Studio
SLIDE_COUNT=5                # количество слайдов
MODE=polling                 # polling | webhook
WEBHOOK_URL=https://...      # если webhook
